# Native I18n Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 为 `apps/desktop-macos-native` 建立可扩展的多语言基础设施，支持系统语言自动选择与设置页手动覆盖，并完成首批中英文资源迁移。

**Architecture:** 使用 Apple 原生 `Localizable.xcstrings` 作为字符串资源中心，新增 `AppLanguage` 负责语言偏好与 `Locale` 解析，并在 `FluxDeckNativeApp` 根节点注入 `Locale`。迁移时先稳定语言状态与资源组织，再逐层清理导航、通用文案和模型层拼接文案，最后覆盖复杂页面与测试。

**Tech Stack:** SwiftUI、Foundation、XCTest、xcodebuild、Xcode String Catalog、Markdown
