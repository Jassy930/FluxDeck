# 2026-03-12 README 面向使用者重写

## 目标

将仓库根目录 `README.md` 从偏开发态的简短说明，重写为面向使用者的项目首页。

## 调整内容

- 重新定义 README 的目标读者为“使用者”
- 突出当前主线状态：
  - 原生桌面端优先
  - `fluxd` / `fluxctl` 可用
  - Web 桌面端仍保留但不是当前优先级
- 增加当前能力说明：
  - Provider / Gateway 管理
  - OpenAI / Anthropic 兼容链路
  - 请求日志与流量指标
  - 运行中 Gateway 更新后自动重启
- 重组快速开始章节，保留最短启动路径
- 调整文档入口为“下一步该看什么”

## 结果

- README 更接近产品首页，而不是仓库维护者笔记
- 新使用者进入仓库后，可以更快建立对项目能力和当前阶段的正确预期
